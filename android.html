Redpin Android Client

How to install

  • Check out the 'RedpinAndroid' and 'RedpinCore' project from the Redpin sourceforge site into your Eclipse workspace.
  • Download the latest Android SDK and ADT Plugin for Eclipse
  • Run the repin client
  • On the Android phone, change the app settings and adjust ip and port to your values (or change the default values in the source file ConnectionHandler.java)

Now you're ready to use Redpin. Have fun!
