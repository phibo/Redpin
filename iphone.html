Redpin iPhone Client

How to install

  • Check out the 'RedpiniPhone' project from the Redpin sourceforge site or download the RedpiniPhone.zip file and open it in XCode
  • Join the iPhone Developer Program
  • Create your App ID and change the XCode project settings accordingly
  • Create a Development Provisioning Profile and install it
  • Build and distribute the Redpin iPhone app
  • On the iPhone, change the app settings and adjust ip and port to your values (or change the default values in the source file ServerConnection.m)

Now you're ready to run Redpin. Have fun!
Feel free to subscribe to our mailing list.
